BinaryNinja Keil Map Loader
===
Usage: `Ctrl + P > Load Keil Map`

# .map format be like
`Image Symbol Table` section is what this plugin focus on.
```
Component: ARM Compiler 5.xx update x (build x) Tool: armlink [<sha>]

==============================================================================

Section Cross References
...

==============================================================================

Image Symbol Table
...

==============================================================================
```
